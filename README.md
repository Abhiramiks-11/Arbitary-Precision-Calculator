# Arbitrary Precision Calculator

## Description
This project is a C program that implements an **Arbitrary Precision Calculator** capable of performing arithmetic operations on numbers larger than the range supported by standard data types.  
The program represents numbers using **doubly linked lists**, allowing digit-by-digit processing for accurate computation of very large integers.

## Features
- Perform addition of very large numbers.
- Perform subtraction of large integers.
- Perform multiplication using digit-wise operations.
- Perform division of large numbers.
- Uses **doubly linked lists** for efficient storage and manipulation of digits.
- Command-line based interface for executing operations.
- Handles numbers beyond standard integer limits.
